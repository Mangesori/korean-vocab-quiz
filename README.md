# Dalkom Korean (달콤한 한국어)

**Dalkom Korean**은 한국어 어휘 학습을 위한 AI 기반 퀴즈 애플리케이션입니다.  
선생님이 단어를 입력하면 AI가 문제를 자동 생성하고, 학생들은 링크 하나로 퀴즈를 풀 수 있습니다.

---

## 주요 기능 (Features)

### 퀴즈 생성 (AI 문제 자동 생성)

선생님이 단어 목록을 입력하면 AI가 문맥에 맞는 문제를 자동으로 생성합니다.

- **단어 입력**: 쉼표(,) 또는 줄바꿈으로 단어를 구분해 입력
- **난이도 선택**: A1 ~ C2 총 6단계 (CEFR 기준)
- **번역 언어**: 영어, 중국어(간/번체), 일본어, 베트남어, 태국어, 인도네시아어, 스페인어, 프랑스어, 독일어, 러시아어 지원
- **세트 구성**: 세트당 단어 수를 1~10개로 조절해 단계별 학습 가능
- **타이머**: 세트별 제한 시간(10초~5분) 설정 가능
- **AI 모델 선택**: OpenAI GPT, Claude, Gemini 중 선택 가능

---

### 퀴즈 유형 (3가지)

퀴즈는 세 가지 유형으로 구성되며, 선생님이 원하는 유형을 조합해 출제할 수 있습니다.

#### 1. 빈칸 채우기 (Fill in the Blank) — 기본 포함
- AI가 생성한 예문에서 핵심 단어를 빈칸으로 제시
- 힌트(단어와 결합할 문법 구조) 제공
- 기본적으로는 번역문이 가려져 있으며 클릭 시 확인 가능
- 예문 음성 재생 지원

#### 2. 문장 만들기 (Sentence Making) — 선택
- 주어진 단어를 활용해 자유롭게 문장을 작성하는 서술형 문제
- AI가 학생의 답안을 자동으로 채점하고 피드백 제공

#### 3. 말하기 연습 (Speaking Practice) — 선택
- 예문을 보고 읽거나, 음성을 듣고 따라 말하며 녹음하는 문제
- AI 발음 평가로 정확도 점수 제공

---

### 퀴즈 관리 (선생님용)

- **문제 편집**: 생성된 예문, 힌트, 번역 직접 수정 가능
- **음성 생성**: 예문에 TTS 음성 자동 생성
- **미리보기**: 학생 화면과 동일하게 퀴즈 미리보기 가능
- **중복 생성**: 기존 퀴즈를 복사해 변형 가능
- **결과 확인**: 학생별 점수, 오답, 소요 시간 등 상세 결과 조회

---

### 공유 및 배포

- **링크 공유**: 로그인 없이 퀴즈를 풀 수 있는 공유 링크 생성
- **응시 횟수 제한**: 공유 링크별 최대 응시 횟수 설정
- **만료일 설정**: 링크 유효 기간 지정 가능
- **익명 응시 지원**: 이름만 입력하면 로그인 없이 퀴즈 응시 가능

---

### 클래스 관리 (Class)

- **클래스 생성**: 선생님이 클래스를 만들고 초대 코드로 학생 초대
- **퀴즈 할당**: 클래스 단위로 퀴즈 배포
- **학생 현황**: 클래스별 학생 목록 및 학습 이력 확인
- **공지사항**: 클래스 내 공지사항 게시 가능

---

### 오답 노트 & 복습

- **오답 노트**: 틀린 문제를 자동으로 모아 확인
- **오답 퀴즈**: 틀린 단어만 모아 다시 퀴즈 생성
- **복습 연습**: 오답 단어로 별도의 연습 세션 진행

---

### 대시보드

- **학생 대시보드**: 최근 퀴즈 결과, 학습 통계, 할당된 퀴즈 확인
- **선생님 대시보드**: 클래스 현황, 학생 활동 현황, 알림 관리

---
